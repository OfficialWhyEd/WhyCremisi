## 1. Brand Philosophy

WhyCremisi is a professional tool for professional musicians. The visual identity reflects this: **no decoration, no noise** вҖ” only intentional elements with a precise function.

```
  CREMISI RED    вҖ” Action В· Danger В· Power В· Precision
  AMBER GOLD     вҖ” Warmth В· Sound В· Organic В· Suggestion  
  DEEP BLACK     вҖ” Control В· Focus В· Mastery В· Silence
```

The name "WhyCremisi" is always written as a single word, capital W and C. Never "Why Cremisi", never "whycremisi", never "WHYCREMISI".

## 8. Animation Principles

```
  Philosophy:   Everything that moves has a reason to move.
                No decorative animation.

  Timing:       Fast responses  вҖ” 150ms ease-out
                State changes   вҖ” 200ms ease-in-out
                Data updates    вҖ” 100ms linear
                Mascot idle     вҖ” 2000вҖ“4000ms ease-in-out (organic)

  Easing:       cubic-bezier(0.16, 1, 0.3, 1)  вҖ” spring-like snaps
                ease-in-out                      вҖ” smooth state shifts
```

## 9. Voice and Tone

```
  вң“  Direct. Precise. No filler.
  вң“  Technical but not cryptic вҖ” translates jargon when helpful.
  вң“  Confident suggestions, never apologetic.
  вң“  Short sentences in chat. Maximum 2 lines before a break.

  вң—  Never says "I think", "maybe", "perhaps".
  вң—  Never repeats what the user just said.
  вң—  Never adds "Let me know if you have any questions."
  вң—  No emoji in professional contexts (only in casual/idle states).
```
