What you can do

No limits.
If you can say it,
it can do it.

There are no supported commands. Just describe what you want — in the language of music, emotion, and sound — and WhyCremisi figures out which knobs to move.

Mixing
"Mix this track the way Chris Lord-Alge would"
Aggressive parallel compression on drums, mid-forward guitars, punchy low end. CLA's signature applied to your loaded plugins — parameter by parameter.
Mastering
"Master this like Stefano Pretolesi"
Clean limiting, controlled low end, Italian hi-fi warmth. Applied to whatever's on your master chain right now.
DAW Control
"Mute the kick and bass — I want to hear the mids only"
Mutes two tracks by name. Transport control, fader automation, solo, record — all native DAW commands.
Phase & Analysis
"Check phases across all tracks, fix anything that's cancelling"
Analyzes polarity relationships between plugins and tracks. Flags and corrects phase cancellations before they kill your low end.
Composition
"Write a full song — verse, chorus, bridge. Dark pop, 128 BPM, A minor"
Opens the piano roll and composes the entire arrangement. Every part. Every note velocity and length. Without you touching a key.
Memory
"What did you do to the master bus 5 minutes ago?"
Queries the JSONL session log. Full recall of every parameter touch, every decision — this session and every previous one.
Native plugins
"Use Logic's stock compressor on the drums, push the attack"
Native DAW plugins are first-class citizens. Logic stock EQ, Ableton's built-in compressor, bundled instruments — all mapped, all controllable exactly like Serum or FabFilter Pro-Q 4. And because it reads VST3 parameter indices at runtime, any plugin released today — Serum 2, Ozone 11, Neutron 5 — works on day zero. No update required.

Philosophy

Built around one idea:
creativity has no limits.

01 — ASK ANYTHING

The tool speaks your language

Most audio software asks you to learn its language — menu structures, preset names, parameter ranges. WhyCremisi flips this. You speak in the language of music and emotion. The software figures out the rest.


You shouldn't need to know that "filter cutoff" is parameter index 0047 in Serum. You should be able to say "make it darker" and have it happen.

02 — MEMORY

It remembers everything

Every parameter touch, every AI decision, every session. Not just this session — every session. It knows your history and learns your taste.

03 — PERSONALITY

A collaborator, not a toolbar

It thinks, replies, explains its reasoning, makes suggestions, pushes back when something doesn't make sense. It has a face that shows you how it feels.

04 — ALWAYS PRESENT

In the room with you

Not an external app, not a chatbot tab. It lives inside your DAW, on the master channel, watching and listening to everything that happens in the session.

05 — FULL AGENT

Web search, bash, code — inside a plugin

WhyCremisi runs a complete AI agent with web search, bash execution, code generation, and file system access — from inside a single VST3 slot.

Under the hood

Built on real
engineering.

⚡

33ms broadcast loop

JUCE Timer fires every 33ms (~30fps), pushing DAW state to every connected React client. Transport, meter L/R/peak — always live.

🧠

JSONL Flight Recorder

Every event appended to events.jsonl with ms timestamps. Cross-session memory.json persists knowledge between sessions.

🌐

WebSocket bridge

OSC UDP :9000/:9001 ↔ WebSocket TCP :8080 ↔ React UI. Typed event system, auto-reconnect, pending request map.

🎛

6 AI providers

Ollama offline by default (llama3.2, localhost:11434). Groq, Gemini, Claude, OpenAI, OpenRouter — swap with one config change.

📦

Dynamic widget system

C++ broadcasts ui.widget.create/update/remove. React renders interactive suggestion widgets live as the conversation evolves.

🔌

VST3 · AU · Standalone

One JUCE 7 codebase, three build targets. 14 automated tests. macOS Monterey+. Windows port on the roadmap.

Compatible with everything

Reads VST3 parameter indices at runtime — any plugin works on day zero.
