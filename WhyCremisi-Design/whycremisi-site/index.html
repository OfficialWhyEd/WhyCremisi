Architecture in 3 Steps

How it actually works

No black boxes. Three concrete layers handle every keystroke from your plugin's chat UI to real-time audio parameter changes.

01 / 03
React UI Layer

React 19 with framer-motion 12 runs inside the JUCE WebView. WebSocket client sends JSON commands, receives streamed AI tokens, drives BotFace animation states in real-time.

port 8765
02 / 03
OscBridge C++

Stateful C++17 bridge process. Owns the WebSocket server, routes messages to 6 AI provider adapters, translates NLP intent into OSC-style AudioProcessor parameter mutations.

C++17 / JUCE OSC
03 / 03
AudioProcessor

Host-synchronized DSP runs in the realtime thread. Receives parameter mutations from the bridge on the message thread, applies them sample-accurately using JUCE's atomic parameter system.

Sample-accurate
9 Emotional States

BotFace reads the room

Morphing SVG paths via Web Animations API. Every state communicates real plugin status — not decoration.

6 AI Providers

Local or cloud. Your call.

Hot-swap providers without reloading the plugin. Every provider shares the same adapter interface — zero abstraction leaks.

OLLAMA
LOCAL · OFFLINE

Run any Llama, Mistral, Gemma model locally. Zero API costs, complete data sovereignty, works air-gapped.

Ollama

Stream tokens from a local model server. No network dependency in the studio. Plugin auto-detects running Ollama instance on localhost:11434.

adapter: OllamaProvider.cpp · streaming: SSE · auth: none
Anthropic Claude

Claude 3.5 Sonnet/Haiku via Messages API. Extended thinking mode for complex harmonic analysis. Streaming with tool use support.

adapter: AnthropicProvider.cpp · model: claude-3-5-sonnet · streaming: SSE
CLAUDE
ANTHROPIC · CLOUD

Extended thinking. Tool use. Best reasoning quality for complex music theory questions.

OPENAI
OPENAI · CLOUD

GPT-4o with function calling. Fine-tuned models supported. Realtime API roadmap integration planned.

OpenAI GPT

GPT-4o multimodal streaming. Plugin sends structured JSON tool schemas to the model. Model returns parameter mutations as structured JSON, bridge validates schema.

adapter: OpenAIProvider.cpp · model: gpt-4o · streaming: SSE
OpenRouter

Unified gateway to 50+ models. Mix providers per session. Pay-per-token across Mistral, Cohere, Perplexity, Command R+, and more.

adapter: OpenRouterProvider.cpp · routing: dynamic · models: 50+
OR
OPENROUTER · GATEWAY

Single API key. Route to any model. Budget limits enforced per session.

GROQ
GROQ · ULTRA-FAST

LPU inference. 500+ tokens/sec. First-token latency under 200ms. Ideal for real-time workflows.

Groq LPU

Language Processing Unit hardware runs Llama 3.1 at 500+ tok/s. Fastest streaming response times of any provider. Plugin BotFace stays in typing state for 180ms average.

adapter: GroqProvider.cpp · model: llama-3.1-70b-versatile · ttft: <200ms
Google Gemini

Gemini 1.5 Pro with 1M token context. Long audio analysis sessions. Multimodal — paste a chord chart screenshot, get back a progression analysis.

adapter: GeminiProvider.cpp · model: gemini-1.5-pro · context: 1M tokens
GEMINI
GOOGLE · MULTIMODAL

1M token context. Multimodal input. Best for long session analysis.
