## Zero-Dependency Installer — Full Specification

Core principle: anyone who receives WhyCremisi has everything working at first launch. No additional downloads. No manual configuration. No external dependencies. Works offline.

### Installer Bundle Contents

`
  ┌──────────────────────────────────────────────────────────┐
  │  CORE PLUGIN                                             │
  │  WhyCremisi.vst3          macOS / Windows / Linux        │
  │  WhyCremisi.component     AU — macOS / Logic only        │
  │  WhyCremisi.app           Standalone macOS               │
  │  WhyCremisi.exe           Standalone Windows             │
  │  WhyCremisi               Standalone Linux               │
  ├──────────────────────────────────────────────────────────┤
  │  LOCAL AI ENGINE (no internet required)                  │
  │  Ollama runtime           binary for each OS             │
  │  Base AI model            llama3.2:3b (~2 GB)            │

### Platform Installers

`
  macOS
    Format:       WhyCremisi-v1.0.pkg
    Installs to:  /Library/Audio/Plug-Ins/VST3/WhyCremisi.vst3
                  /Library/Audio/Plug-Ins/Components/WhyCremisi.component
                  /Applications/WhyCremisi.app
    Signing:      Apple Developer ID signed + notarised
    Requirements: macOS 12 Monterey+, Apple Silicon or Intel

  Windows
    Format:       WhyCremisi-v1.0-setup.exe  (NSIS / Inno Setup)
    Installs to:  C:\Program Files\Common Files\VST3\WhyCremisi.vst3
                  C:\Program Files\WhyCremisi\WhyCremisi.exe
    Signing:      EV code signing certificate
    Requirements: Windows 10 64-bit+

  Linux
    Formats:      whycremisi-v1.0.tar.gz + install.sh
                  whycremisi-v1.0.deb  (Debian/Ubuntu)
                  whycremisi-v1.0.rpm  (Fedora/RHEL)
    Installs to:  ~/.vst3/WhyCremisi.vst3
    Requirements: Ubuntu 22.04+ / Fedora 38+
`

### First Launch Flow

`
  1. Plugin loaded in DAW (or Standalone opened)
  2. SetupScreen appears automatically
  3. Choice: local AI (Ollama — already bundled) or cloud API
  4. If local: Ollama starts automatically (no download needed)
  5. Automatic connection test
  6. Ready to use in < 30 seconds

  NO additional downloads
  NO mandatory registration
  NO external dependencies
  WORKS fully offline (with local AI)
`
